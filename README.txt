Описание проект "success way".
--------------------------------------------------------------------------------------------------------------------------------
Приложение построено на ASP.NET MVC Framework
---------------------------------------------
Умеет:
Принимать файлы с расширениями (txt/docx) и делать шифровать/дешифровать их содержимое.
Для каждой задачи предусмотрена отдельная страница:
	- Зашифровать TXT
	- Расшифровать TXT
	- Зашифровать DOCX
	- Расшифровать DOCX

Каждая из страниц имеет схожий интуитивно - понятный интерфейс.
--------------------------------------------------------------------------------------------------------------------------------
Логика работы:
	1) Нужно загрузить файл.
		После загрузки вы увидите следующее: "Файл готов к шифрованию!"
		И сможете посмотреть исходный текст этого файла.
		------------------------------------------------
		При загрузке файла формата txt  - файл сохраняется с именем "encrypt.txt"  / "decrypt.txt" в папке "Files", которая находится в корне проекта
		При загрузке файла формата word - файл сохраняется с именем "encrypt.docx" / "decrypt_word.docx' в папке "Files", которая находится в корне проекта
		Эти файлы используются в методах для шифровки.
	2) Ввести ключ и нажать кнопку "Зашифровать/Дешифровать"
		------------------------------------------------
		При шифровании/дешифровании файла формата txt  - в папке "Files" создается файл "encrypt_out.txt" / "decrypt_out.txt"
		При шифровании/дешифровании файла формата word - в папке "Files" создается файл "encrypt_out.docx"/ "decrypt_word_out.docx'
		Эти файлы используются в методах для шифровки.
	3) После нажатия кнопки появится кнопка "скачать" и так же можно посмотреть результат операции из пункта 2.
	4) При нажатии кнопки "Скачать" вы выбираете путь куда сохранить файл.
		При этом скачивается соответствующий файл "encrypt_out.txt" / "decrypt_out.txt" || "encrypt_out.docx"/ "decrypt_word_out.docx'
		Исходный и результат файлы удаляются (вне зависимости от выбора пользователя "сохранить/отмена")

--------------------------------------------------------------------------------------------------------------------------------
Основная составляющая проекта (шифровка/дешифровка) покрыта юнит тестами.
